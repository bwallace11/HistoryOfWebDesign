# HistoryOfWebDesign
A walk through on the history of web design and the development.
WARNING: This project intentionally demonstrates historically accurate web technologies. 
Do not modernize earlier eras.
[Live Site](https://github.com/bwallace11/HistoryOfWebDesign)

## Entry files
- `index.html` is the primary app entry point used by static hosts and direct folder browsing.
- `index_new.html` is an alternate draft/backup shell and is not loaded unless opened directly.
