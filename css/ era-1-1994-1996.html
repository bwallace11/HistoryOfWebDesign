1994–1996: CSS Is Proposed (Before CSS1 Is Official)

What’s going on in this era?

Early web pages were mostly plain text and links. Authors wanted control over fonts, spacing, and layout, but HTML did not have a clean styling system. In 1994, Håkon Wium Lie proposed the concept of Cascading Style Sheets while working in the early Web community (connected to CERN and W3C work). :contentReference[oaicite:1]{index=1}

Why this mattered
  • HTML was getting “presentational” fast (tags like <font> and alignment tricks).
  • Sites needed consistent styling without repeating formatting on every element.
  • A “cascade” meant multiple style sources could interact (browser defaults, user preferences, author styling).
Major players
  • Håkon Wium Lie (proposed CSS in 1994). :contentReference[oaicite:2]{index=2}
  • Bert Bos (co-inventor, joined W3C in the mid-1990s and worked on CSS). :contentReference[oaicite:3]{index=3}
  • World Wide Web Consortium (W3C) (the standards organization coordinating CSS work). :contentReference[oaicite:4]{index=4}
Important notes
  1. CSS starts as a proposal before it becomes a formal W3C Recommendation.
  2. Early browsers were inconsistent, so HTML “formatting tags” kept spreading anyway.
