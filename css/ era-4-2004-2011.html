2004–2011: CSS becomes modular, and CSS 2.1 becomes the stable base

Main shift: CSS stops being “one huge spec” and becomes a set of modules (often called “CSS3” modules). Meanwhile, CSS 2.1 becomes the stable baseline used for consistent implementation.

What to know

  • The CSS Working Group focuses on clarifying and stabilizing behavior so browsers can match each other better. :contentReference[oaicite:14]{index=14}
  • “CSS3” is not one single version that ships at once; it is a modular approach. :contentReference[oaicite:15]{index=15}
  • This era sets up the modern world: better interoperability, better docs, better testing culture.

Major players

  • W3C CSS Working Group (CSSWG) coordinating specs and tests. :contentReference[oaicite:16]{index=16}
  • Editors and implementers across browser companies and the W3C staff network.

Era CSS rule: CSS2.1-style basics only. No media queries, no flexbox, no grid.
