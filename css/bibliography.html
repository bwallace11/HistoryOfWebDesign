Bibliography: Free, credible CSS history sources

These are primary specs and reputable references you can cite. All links are free to read.

W3C (World Wide Web Consortium) specifications (primary sources)

  • CSS1 (Cascading Style Sheets, level 1) W3C Recommendation (historical versions are on W3C): W3C CSS1 (1996 proposal/reports)
  • CSS2 (Cascading Style Sheets, level 2) W3C Recommendation (May 1998): W3C CSS2 Recommendation
  • CSS 2.1 (Cascading Style Sheets Level 2 Revision 1) W3C Recommendation (June 2011): W3C CSS 2.1 Recommendation
  • CSS Snapshot 2022 (explains modern “modules,” and that there is no single CSS Level 4): W3C CSS Snapshot 2022
  • CSS Grid Layout Module Level 1 (current report): W3C CSS Grid Layout Level 1
  • CSS Cascading and Inheritance Level 5 (cascade layers info): W3C CSS Cascade Level 5
  • Selectors Level 4 (modern selector work, including :has()): W3C Selectors Level 4

Trusted documentation and reference

  • MDN Web Docs (Mozilla Developer Network): CSS Cascade guide: MDN: Cascade
  • Library of Congress format description (useful for a “credible institution” citation): Library of Congress: CSS format overview

Contributor reference pages (quick access)

  • Wikipedia: Håkon Wium Lie
  • Wikipedia: Bert Bos
  • Wikipedia: Tim Berners-Lee
  • Wikipedia: Chris Lilley

Tip for school writing: specs (W3C) are your strongest sources. Use Wikipedia only as a starting point, then cite specs and official org pages.
