Major Contributors to CSS

Håkon Wium Lie

Proposed cascading style sheets in October 1994 while working at CERN with Tim Berners-Lee. His proposal, "Cascading HTML Style Sheets," introduced the cascade mechanism and the idea of author-versus-reader style priorities. Later became CTO of Opera Software.

Bert Bos

Co-designed CSS with Håkon Wium Lie and became a primary editor of CSS specifications at W3C. Has been central to the W3C CSS Working Group for decades.

Tantek Çelik

A key figure in CSS 2.1 editing and a long-time browser implementer who discovered and documented many cross-browser CSS bugs. Advocated for web standards adoption during the "browser wars" cleanup era.

Elika J. Etemad (fantasai)

A prolific CSS specification editor who has worked on dozens of CSS modules. Instrumental in shaping the modern modular CSS architecture and ensuring specification precision.

Rachel Andrew & Jen Simmons

Both played major roles in educating developers about CSS Grid Layout and modern layout techniques, making complex specifications accessible to working developers.

W3C CSS Working Group

The committee responsible for all CSS specifications, composed of browser vendor representatives and invited experts. Meets regularly to advance CSS modules from proposal to Recommendation.
