CSS Contributors (Major People)

Håkon Wium Lie

Håkon Wium Lie proposed the idea of Cascading Style Sheets in 1994, when the Web badly needed a real styling system. He’s one of the central founders of CSS thinking.

  • W3C page: https://www.w3.org/People/howcome/
  • Wikipedia: https://en.wikipedia.org/wiki/H%C3%A5kon_Wium_Lie

Bert Bos

Bert Bos is a co-inventor of CSS and worked at W3C on CSS and other standards work. He collaborated early with Lie and helped shape CSS into an implementable standard.

  • W3C page: https://www.w3.org/People/Bos/
  • W3C staff bio: https://www.w3.org/staff/?initial=b
  • Wikipedia: https://en.wikipedia.org/wiki/Bert_Bos

CSS Working Group (CSSWG) contributors (ongoing)

CSS is maintained and expanded by the Cascading Style Sheets Working Group at W3C, made up of browser engineers, spec editors, and invited experts.

  • CSSWG participants list: https://www.w3.org/Style/CSS/members
  • CSSWG charter: https://www.w3.org/2019/10/css-wg-charter.html

Note: Modern CSS is maintained by many editors across many modules. The participants list is the most accurate “current roster.”
