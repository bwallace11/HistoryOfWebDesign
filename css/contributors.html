Major Contributors to Cascading Style Sheets (CSS)

This page documents the individuals, editors, working group members, and browser engineers who significantly shaped CSS from 1994 to today.

Håkon Wium Lie

Proposed CSS in 1994 while working with the World Wide Web Consortium (W3C). He envisioned separating document structure (HTML) from presentation. He later worked at Opera, advocating strongly for standards compliance.

Contributions:

  • Co-author and editor of CSS1 and CSS2
  • Shaped the cascade model
  • Helped define media types

Reference: Wikipedia

Bert Bos

Co-developer of CSS and long-time W3C editor. Bos collaborated on CSS1 and CSS2 and played a central role in refining CSS 2.1.

  • CSS1 and CSS2 co-editor
  • Key contributor to CSS2.1 stabilization
  • Advocate for modular CSS development

Reference: Wikipedia

Chris Lilley

CSS2 editor and W3C contributor focused on typography and graphics standards. Played a key editorial role during CSS’s expansion into layout.

Reference: Wikipedia

Women Who Shaped CSS and Web Standards

Tantek Çelik

Early Internet Explorer engineer and later Web standards advocate. Helped improve standards compliance and worked closely on CSS implementations.

Anne van Kesteren

Long-time standards contributor and editor in web platform technologies. Contributed to specifications and browser engine alignment affecting CSS behavior.

Fantasai (Elika Etemad)

Editor in the W3C CSS Working Group, heavily involved in CSS specifications including layout modules and text-level details.

Tab Atkins Jr.

CSS Working Group co-chair and editor for multiple modern CSS modules, including Grid, Cascade, and Selectors.

CSS evolution has depended on many contributors inside browser vendors and W3C, including women and underrepresented engineers who worked through specification editing and test suite development.
