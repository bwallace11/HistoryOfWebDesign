Founders and Early Architects (1994–1998)

  • Håkon Wium Lie
    Co-creator of CSS. In 1994, while working at CERN, he proposed the idea of a separate stylesheet language that would allow authors to control layout and typography without modifying HTML. He introduced the concept of "cascading," where rules are prioritized by origin and specificity. Later joined Opera Software and continued advocating for open standards.

  • Bert Bos
    Co-creator of CSS and one of its longest-serving editors. He collaborated with Lie to formalize early CSS concepts and became editor of CSS Level 1 and CSS Level 2. Bos has remained deeply involved in CSS Working Group activities and helped define structural behavior of the language, including selectors and formatting models.

  • Tim Berners-Lee
    Founder of the World Wide Web and Director of the World Wide Web Consortium (W3C). While not a CSS author, he supported the separation of content (HTML) and presentation (CSS), which became a foundational architectural principle.

CSS Level 2 and Standards Leadership (1998–2005)

  • Chris Lilley
    Editor of CSS2 (1998). Worked extensively on color standards, web fonts, and graphical aspects of CSS. Contributed to SVG and CSS integration efforts.

  • Ian Jacobs
    Co-editor of CSS2 and long-time W3C staff member. Helped coordinate specification publication and standards process management.

  • Tantek Çelik
    Editor of CSS 2.1. Key contributor during the browser wars era. Worked to reconcile browser inconsistencies and stabilize CSS behavior into what became the widely implemented CSS 2.1 baseline.

  • Ian Hickson
    Editor of CSS 2.1 and major specification editor across web technologies. Focused on clarifying ambiguous behavior and improving interoperability across engines.

Modern CSS Editors and Specification Leaders (2005–Present)

  • Elika J. Etemad (fantasai)
    One of the most influential modern CSS editors. Lead editor on numerous modules including CSS Grid and CSS Logical Properties. Actively guides interoperability efforts in the CSS Working Group.

  • Tab Atkins Jr.
    Specification editor for many modern CSS modules including Flexbox, Selectors Level 4, and various layout-related updates. Strong advocate for clear spec writing and implementer communication.

  • Lea Verou
    Researcher and CSS Working Group contributor. Advanced tooling, selector development, and developer education. Author of influential CSS resources and widely recognized educator.

  • Miriam Suzanne
    Advocate for modern layout systems and design systems. Contributed to the evolution of container queries and layout philosophy.

Educators and Layout Advocates

  • Rachel Andrew
    Major educator and author on CSS Grid. Wrote key documentation and worked with browser vendors to promote understanding of modern layout patterns.

  • Jen Simmons
    Mozilla advocate who championed CSS Grid. Produced educational material and live coding demonstrations that accelerated Grid adoption.

  • Nicole Sullivan
    Pioneer of scalable CSS architecture (Object-Oriented CSS). Focused on modularity and performance.

Browser Engine Teams (Implementation Power)

CSS does not matter if browsers do not implement it. Engine teams shaped real-world CSS behavior.

  • Trident — Microsoft Internet Explorer engine (1997–2015). Defined much early layout behavior.
  • Gecko — Mozilla engine. Strong standards alignment and test-driven implementation.
  • WebKit — Engine powering Safari; major influence on mobile CSS evolution.
  • Blink — Chromium engine; accelerated modern CSS shipping cycles.
  • Presto — Opera engine known for strong standards adoption in the 2000s.
