CSS History (1994–2026)

CSS means Cascading Style Sheets. It started in 1994 because the early Web had no real way to style documents. The first big idea came from Håkon Wium Lie, later joined by Bert Bos.

Eras (separate pages)

  1. 1994–1996: The CSS idea appears and CSS1 forms
  2. 1996–1998: CSS1 becomes a W3C Recommendation, CSS2 arrives
  3. 1998–2004: Browser wars, broken support, and survival
  4. 2004–2011: CSS3 modules begin; CSS 2.1 becomes the stable base
  5. 2011–2017: Modern layout becomes real (media queries, flexbox, etc.)
  6. 2017–2026: Grid era and the new wave (variables, container queries, :has)

Why CSS exists

CSS exists because people wanted control over visual design without turning HTML into an unreadable pile of formatting tags.
