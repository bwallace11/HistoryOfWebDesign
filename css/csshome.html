Cascading Style Sheets (CSS) History

This site explains the history of Cascading Style Sheets (CSS) from the first proposals in the mid-1990s to modern modular CSS (through 2026).

Era map (CSS-focused)

  1. 1994-1996: Before CSS1 (proposals + early experiments)
  2. 1996-1998: CSS Level 1 (CSS1) and the first real implementations
  3. 1998-2001: CSS Level 2 (CSS2), positioning, media types, early standard pain
  4. 2001-2011: CSS 2.1 stabilization, "browser wars" aftermath, baseline solidifies
  5. 2011-2017: CSS3 as modules (media queries, flexbox, transforms), modern layout begins
  6. 2017-2026: Grid, variables, container queries era, cascade layers, modern selectors

Go in order if you want the "why it turned out like this" story.

This section documents each era using only the HTML and CSS available during that time period.
