Cascading Style Sheets: Era 1 (1990–1996)

CSS stands for Cascading Style Sheets. This era covers the conditions that created CSS and the first real proposal work that led to CSS Level 1.

Context: why CSS had to exist

Early web pages were mostly structural markup (headings, paragraphs, lists, links). As the web expanded past research, authors wanted control over type, spacing, colors, and basic layout. Browsers responded by adding presentational HTML features (for example, font tags and alignment) but that mixed content and presentation and made sites hard to maintain. CSS was created as a separate, reusable styling layer to prevent HTML from turning into a formatting mess.

Key milestones (1990–1996)

  • 1990–1993: The World Wide Web spreads from research to public networks; authors want more control.
  • 1994: Early CSS proposals circulate; the goal is to style HTML documents without changing HTML itself.
  • 1994–1996: CSS develops through discussion, prototypes, and standards work, preparing for CSS1.
  • 17 December 1996: CSS Level 1 becomes a World Wide Web Consortium recommendation. (See bibliography.)

Major people and institutions

  • Håkon Wium Lie (CERN): proposed early cascading style sheets and pushed the idea as the web grew.
  • Bert Bos: co-designed early CSS and became a major editor of CSS specifications.
  • Tim Berners-Lee (CERN / W3C): founded the World Wide Web Consortium to coordinate web standards.
  • World Wide Web Consortium (W3C): the standards organization that published CSS recommendations.

What shipped (reality check)

Before CSS1 was a stable recommendation, browser support was limited and inconsistent. Authors relied mostly on HTML presentation features and browser defaults.

What authors actually did (1990–1996)

  • Presentational HTML: authors used tags and attributes because CSS support was not reliable.
  • Defaults and patience: pages leaned on browser default fonts and spacing.
  • Early experiments: tiny embedded style sheets or testing pages, usually not production-critical.

This era ends when CSS Level 1 becomes a W3C recommendation on 17 December 1996.
