1996–1998: CSS1 becomes official, CSS2 follows

What happened

CSS Level 1 became a W3C Recommendation in December 1996. :contentReference[oaicite:6]{index=6} Then CSS Level 2 arrived in May 1998, expanding what CSS could describe. :contentReference[oaicite:7]{index=7}

Why CSS1 mattered

  • It gave authors a real way to set fonts, text styles, colors, and spacing without stuffing HTML with formatting tags.
  • It introduced the core idea of the cascade (rules competing by priority).
  • It created a real target for browser implementers, even if support was messy.

Why CSS2 mattered

  • It expanded layout and media concepts (more device types, stronger layout goals).
  • It pushed CSS toward being capable of replacing many HTML “layout hacks.”

Major players (this era)

  • Håkon Wium Lie and Bert Bos (core creators/editors). :contentReference[oaicite:8]{index=8}
  • W3C as the standards home for CSS work. :contentReference[oaicite:9]{index=9}
  • Browser vendors implementing partial CSS: Netscape, Microsoft, Opera (support differed a lot).
