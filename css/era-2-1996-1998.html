Key Event

On December 17, 1996, the W3C published Cascading Style Sheets, level 1 (CSS1) as a Recommendation. It defined basic styling: fonts, colors, text formatting, margins, padding, borders, and simple selectors.

Major People and Groups

  • Håkon Wium Lie – CSS1 author/editor
  • Bert Bos – CSS1 author/editor
  • W3C CSS Working Group – produced the Recommendation

Browser Reality (Why It Was Hard)

CSS existed on paper, but browsers implemented it inconsistently. Authors could not rely on it for layout. This is why presentational HTML kept thriving even when CSS was “official.”

What Authors Actually Did

  • Used CSS mostly for fonts and colors, not layout.
  • Kept <font> and <center> as “backup.”
  • Kept layouts in tables and used CSS as optional enhancement.
  • Used inline styles when they worked, because external styles sometimes failed.

CSS1 Example (Era-Appropriate)
