Era 2: CSS Level 1 and first real browser support (1996-1998)

What changed in this era

CSS Level 1 (often written as CSS1) becomes an official recommendation on 17 December 1996. This mattered because it created a shared baseline for browsers and authors: a real spec for fonts, colors, backgrounds, text styling, and the basic box model (margin, border, padding).

CSS Level 1 focus:
  • Font properties (family, size, weight, style)
  • Color and background color
  • Text properties (alignment, decoration)
  • Box properties (margin, border, padding)
  • Classification hooks (class and id) for selecting elements

Major players

  • Håkon Wium Lie and Bert Bos: core CSS design leadership.
  • World Wide Web Consortium (W3C): publishes CSS1 recommendation.
  • Browser implementers: early CSS support appears unevenly across products.

What shipped (the messy truth)

CSS existed on paper before it existed reliably in browsers. During 1996-1998, support was partial and buggy. Authors still used HTML presentation features heavily, because those worked.

What authors actually did

  • Inline style and tiny embedded styles: small experiments where failure was acceptable.
  • Fallback-first: assume CSS might fail and content still must be readable.
  • Keep it shallow: simple color/font changes, not complex layout.

Artifact: Internet Explorer 3 CSS Support (1996)

Title: First commercial browser with CSS support
Year: 1996
Browser: Internet Explorer 3

IE3 was the first widely used browser to ship with partial CSS1 support. The implementation was incomplete and buggy, but it proved that CSS could work in practice. Netscape Navigator added CSS support in version 4 (1997), but its implementation was even more inconsistent.

Why it mattered: Having any CSS support in a major browser encouraged authors to experiment. Even partial support showed that the concept of separating style from content could work.

Artifact: Early CSS example code

What CSS1 looked like in practice:

body {
  background-color: #ffffff;
  color: #000000;
  font-family: Arial, Helvetica, sans-serif;
}

h1 {
  font-size: 24pt;
  color: #000080;
}

a:link { color: #0000EE; }
a:visited { color: #551A8B; }

This was the extent of what worked reliably. Font changes, colors, and basic text styling. Layout was still handled entirely by HTML tables.

Limitations designers faced

  • CSS layout properties (float, position) were unreliable across browsers.
  • No way to create equal-height columns with CSS.
  • Box model differences between IE and Netscape caused spacing problems.
  • Authors had to test in multiple browsers and often maintain separate stylesheets.

CSS1 becomes a W3C recommendation on 17 December 1996.
