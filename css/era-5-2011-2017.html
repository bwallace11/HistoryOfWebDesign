Era 5: CSS3 modules, responsive design, modern UI patterns (2011-2017)

CSS becomes modular: instead of "one CSS3," you get stable modules like backgrounds, borders, selectors, media queries, and transforms.

What changes

  • Responsive design becomes mainstream (media queries + flexible layouts).
  • Rounded corners and shadows become normal: border-radius, box-shadow.
  • Visual polish without images: gradients, transforms, transitions.

Major contributors

  • Rachel Andrew: CSS Grid educator and standards communicator.
  • Jen Simmons: major advocate/teacher for modern layout at Mozilla.
  • Lea Verou: CSS educator, tooling, and standards work.
  • Elika J. Etemad (fantasai): major CSS editor and spec leader.

What authors actually did

  • Mobile-first layouts: design for small screens, then add rules for larger screens.
  • "Flat design": minimal decoration, bold type, strong spacing.
  • Frameworks: prebuilt grids and UI patterns become common.
  • Vendor prefixes: write the same property several times (example: -webkit-).
