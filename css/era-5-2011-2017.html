2011–2017: Responsive design becomes standard, layout gets serious

This era is where “modern CSS” starts feeling normal: responsive design is everywhere, and browser updating gets faster. Media Queries become a W3C Recommendation in 2012, which locks in responsive design as a standard tool. :contentReference[oaicite:18]{index=18}

What changes

  • Responsive layouts become expected (mobile is not optional).
  • CSS is modular and grows feature-by-feature instead of “one big CSS3 drop.” :contentReference[oaicite:19]{index=19}
  • Flexbox becomes a mainstream layout tool (used carefully at first).

Major players

  • Browser vendors shipping features faster (Chrome, Firefox, Safari, IE/Edge direction shift).
  • CSS Working Group editors and contributors coordinating modules and tests. :contentReference[oaicite:20]{index=20}
