What Changed

Instead of one giant “CSS3 spec,” CSS evolved as modules. That let features ship independently: selectors, backgrounds, transforms, flexbox, and more.

Major Features (Common by 2017)

  • Media queries (responsive design)
  • Transforms, transitions, animations
  • Gradients and multiple backgrounds
  • Flexbox (modern layout model)

Engine Teams Driving This

  • Blink (Chrome, forked from WebKit in 2013)
  • WebKit (Safari)
  • Gecko (Firefox)
  • EdgeHTML (older Edge, before Chromium switch later)

What Authors Actually Did

  • Vendor prefixes: -webkit-, -moz-, -ms-, -o-
  • Autoprefixing tools became common to manage prefix mess
  • Frameworks (Bootstrap era) shaped default design language
  • Float layouts persisted until flexbox became safe enough
