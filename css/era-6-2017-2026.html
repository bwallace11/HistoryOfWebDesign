Major Features

  • CSS Grid Layout (two-dimensional layout)
  • Container Queries (component-based responsiveness)
  • :has() (parent-aware selection)
  • Cascade Layers (@layer)
  • Modern color and typography improvements

What Authors Actually Did

  • Moved from heavy frameworks to design systems and utility CSS.
  • Reduced prefix hacks because evergreen browsers update automatically.
  • Used grid for page layout, flexbox for components, and containers for responsive blocks.
  • Fought complexity: build tooling, bundlers, and “CSS-in-JS” debates.

Engine Teams (Modern Reality)

  • Blink (Chrome / Chromium browsers)
  • WebKit (Safari)
  • Gecko (Firefox)
