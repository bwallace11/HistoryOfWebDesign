Glossary (CSS terms + acronyms)

CSS (Cascading Style Sheets)
A language for describing how HTML content should look (fonts, spacing, layout, colors).
W3C (World Wide Web Consortium)
The standards organization that hosts working groups for web specs, including CSS.
CSSWG (Cascading Style Sheets Working Group)
The group inside W3C that writes and maintains CSS specifications and related tests.
Recommendation (W3C Recommendation)
A mature W3C standard. It means the spec is stable enough to be treated as a web standard.
Working Draft
An early version of a spec that can change a lot.
Candidate Recommendation
A stage where a spec is asking for implementation experience and testing.
Property
A CSS setting like color, font-size, or margin.
Value
The setting you assign to a property, like color: red;.
Declaration
A single property + value pair: margin: 10px;
Rule
A selector plus declarations: p { color: black; }
Selector
The part that chooses what elements get styled (like p or .class).
Class (HTML class attribute)
A label used to target elements in CSS (example: class="card").
ID (HTML id attribute)
A unique label used to target one element (example: id="header").
Specificity
The “priority math” browsers use when multiple rules compete for the same element.
Cascade
The system that decides which CSS wins when multiple rules apply.
Inheritance
When some styles (like font family) flow from parent elements to children automatically.
Box Model
Every element is treated like a box: content, padding, border, margin.
Media Query
A CSS feature that applies styles only under certain conditions (like screen width). Media Queries Level 3 became a W3C Recommendation in 2012.
Responsive Design
A design approach where layout adapts to screen size and device constraints.
Flexbox (Flexible Box Layout)
A layout system mainly for one-dimensional layouts (row OR column).
Grid (CSS Grid Layout)
A layout system for two-dimensional layouts (rows AND columns).
Custom Properties (CSS Variables)
Author-defined properties like --main-color that can be reused.
Container Query
CSS rules that respond to the size of a container instead of the viewport.
