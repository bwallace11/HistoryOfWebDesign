Glossary and Acronyms (CSS + Web Basics)

This is written for people who are new to HTML and CSS. Terms come from many eras.

Acronyms (spelled out)

CSS
Cascading Style Sheets. The language used to control presentation (fonts, spacing, layout) of web documents.
HTML
HyperText Markup Language. The markup language for document structure and meaning.
W3C
World Wide Web Consortium. A standards organization that publishes web specifications.
IETF
Internet Engineering Task Force. Another standards organization (important earlier for web standards).
URL
Uniform Resource Locator. A web address.
HTTP
HyperText Transfer Protocol. The network protocol used to request and deliver web documents.
SGML
Standard Generalized Markup Language. An older markup standard that influenced HTML.
DOM
Document Object Model. A programming interface for documents. (Mentioned in history even if your site avoids JavaScript.)
UA
User Agent. Usually a web browser, but can be other tools that read web content.
DTD
Document Type Definition. A grammar for SGML/XML documents (important in older HTML/XHTML history).

CSS terms (beginner friendly)

Rule
A chunk of CSS: selector + declarations.
Selector
The part that chooses what elements the rule applies to (example: p, .card, #header).
Declaration
A property and value pair (example: color: black;).
Property
The thing you are changing (color, margin, display, font-size).
Value
The setting for the property (black, 16px, grid, 1.5).
Cascade
The system that decides which CSS wins when multiple rules conflict.
Specificity
A scoring system for selectors. More specific selectors override less specific ones (when other factors tie).
Inheritance
Some properties pass from parent elements to children (like font-family), others do not (like margin).
Box Model
How elements take up space: content, padding, border, margin.
Block
Elements that start on a new line and usually take full width (like div, p).
Inline
Elements that flow within a line of text (like span, a).
Layout
How elements are placed on the page (floats, flexbox, grid).
Float
An older layout trick that let text wrap around elements. Used heavily before flexbox/grid.
Flexbox
Flexible Box Layout. Good for one-direction layouts (row or column) and alignment.
Grid
CSS Grid Layout. Good for two-direction layouts (rows and columns together).
Media Query
A CSS feature that applies rules based on conditions like screen width (responsive design).
Responsive Design
Design that adapts to different screens and devices, often using media queries.
Vendor Prefix
Old browser prefixes like -webkit-, -moz-, used for experimental features.
Custom Property
CSS variable using --name, used with var(--name).
Container Query
Responsive styling based on a container’s size, not the viewport.
Cascade Layer
@layer feature to control which groups of styles win before specificity fights.

This glossary is intentionally big because beginners deserve definitions, not “just google it.”
