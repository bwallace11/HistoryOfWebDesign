Glossary of CSS Terms

Key terms used throughout the CSS history archive.

Cascading

The mechanism by which CSS resolves conflicts when multiple rules apply to the same element. Rules cascade according to specificity, origin (author, user, browser), and order of appearance.

Selector

The part of a CSS rule that identifies which HTML elements the rule applies to. Examples include element selectors (p), class selectors (.note), and ID selectors (#header).

Box model

The CSS model that treats every element as a rectangular box with content, padding, border, and margin areas. Understanding the box model is essential for controlling layout and spacing.

Specificity

A weighting system CSS uses to decide which rule wins when multiple rules target the same element and property. Inline styles beat IDs, which beat classes, which beat element selectors.

Float

A CSS property that removes an element from normal document flow and shifts it to the left or right, allowing text to wrap around it. Before Flexbox and Grid, floats were the primary tool for multi-column layouts.

Flexbox (Flexible Box Layout)

A CSS layout model introduced in CSS3 that provides efficient alignment and distribution of space among items in a container, especially when sizes are unknown or dynamic.

Grid Layout

A two-dimensional CSS layout system introduced widely in 2017 that divides a container into rows and columns, enabling complex layouts without nested floats or tables.

Custom properties (CSS variables)

Variables defined in CSS using the --name: value syntax and referenced via var(--name). Introduced in CSS3, they allow themes and reusable values without a preprocessor.

Media query

A CSS feature that applies rules only when certain conditions are true, typically related to the viewport width. The basis of responsive design, standardized in CSS3.

Vendor prefix

Browser-specific prefixes such as -webkit-, -moz-, and -ms- used during the CSS3 era to experiment with features before standardization. Now largely deprecated in favor of unprefixed properties.
