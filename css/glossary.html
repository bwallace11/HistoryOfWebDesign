Glossary of CSS Terms

Key terms used throughout the CSS history archive.

Cascading

The conflict-resolution system that determines which rule wins by origin, specificity, and source order.

Selector

The rule target syntax, such as element, class, ID, attribute, and relationship selectors.

Box Model

The rectangular model for content, padding, border, and margin used for sizing and spacing.

Specificity

A weighting system used to break ties between competing declarations.

Float

A legacy positioning approach that moved elements left/right and wrapped inline content around them.

Flexbox

A one-dimensional layout model for alignment and spacing across a row or column.

Grid Layout

A two-dimensional layout model for rows and columns with precise track control.

Custom Properties

Native CSS variables declared with --name and consumed with var().

Media Query

Conditional styling based on viewport or environment characteristics.

Vendor Prefix

Legacy browser prefixes used during experimental implementation phases.
