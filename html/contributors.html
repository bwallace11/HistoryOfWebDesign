Major Contributors to HTML

Tim Berners-Lee

Invented HTML and the World Wide Web at CERN around 1990-1991. Wrote the first web browser and web server. Founded the W3C in 1994 and remains its director, guiding web standards for over three decades.

Robert Cailliau

Collaborated with Berners-Lee at CERN to develop and promote the World Wide Web project. Helped secure institutional support and resources for the early Web.

Dave Raggett

Key editor at W3C for HTML 3.0, HTML 3.2, and HTML 4.0. Proposed many features that became foundational to modern HTML, including tables and forms extensions.

Ian Hickson

Principal editor of the WHATWG HTML5 specification for many years. His work defined parsing rules, error handling, and the application-oriented features of HTML5 that shipped in all modern browsers.

WHATWG Founders (2004)

Engineers from Apple (including Maciej Stachowiak), Mozilla (including David Baron and Ian Hickson), and Opera (including Anne van Kesteren and Lachlan Hunt) founded WHATWG to continue pragmatic HTML development when W3C focused on XHTML 2.0.

W3C HTML Working Group

The committee within W3C responsible for HTML standards, including HTML 4.01, XHTML 1.0, and the W3C HTML5 Recommendation. Co-chairs over time included Paul Cotton and Sam Ruby.
