Early HTML: 1990–1994

HTML was created by Tim Berners-Lee at CERN around 1990–1991 to share linked scientific documents on the emerging World Wide Web. It was based on SGML and defined a small set of structural and inline tags for headings, paragraphs, lists, and hyperlinks.

At this stage there was no official standard; HTML existed as informal documentation and code in the first browsers and servers. The focus was on simple hypertext documents, not layout or visual design.

Major contributors and institutions

  • Tim Berners-Lee – designed the first HTML tags, wrote the first web server and browser at CERN.
  • Robert Cailliau – collaborated with Berners-Lee on promoting and refining the early Web project.
  • CERN – the research lab where the Web and HTML were born and first deployed.

Key ideas in early HTML

  • Hyperlinks using the a element and the href attribute to connect documents across machines.
  • Headings h1–h6, paragraphs p, and lists ul, ol, li for basic document structure.
  • No presentational tags: appearance was largely decided by the browser, not the author.
