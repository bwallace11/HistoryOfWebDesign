HTML 2.0, 3.0, and 3.2

Standardizing early web pages and the first browser wars

Version Years Highlights
HTML 2.0 1995 First formal HTML standard from the IETF; defined core tags for forms, images, and basic document structure.
HTML 3.0 (draft) 1995-1996 Ambitious draft led by Dave Raggett, adding tables, math, and more; abandoned because it was too large and browsers could not keep up.
HTML 3.2 1997 First W3C HTML recommendation, code-named "WILBUR"; standardized many features in use, including tables, images, and some presentational elements.

Major contributors

  • Dave Raggett - key editor of early HTML specifications.
  • Lou Montulli - creator of the Lynx text browser and early Netscape engineer.
  • Netscape Communications - introduced many proprietary tags during the browser wars.
  • Microsoft - Internet Explorer team competed with Netscape.
  • W3C (World Wide Web Consortium) - founded 1994 to coordinate standards.

Important changes in this era

  • Introduction of tables that authors quickly used for page layout as well as data.
  • Widespread use of presentational elements such as font, center.
  • Growing tension between browser-specific tags and the need for common standards.
