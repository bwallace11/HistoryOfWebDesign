Acronyms and initialisms

HTML
HyperText Markup Language – the markup language used to structure documents on the World Wide Web.
XHTML
eXtensible HyperText Markup Language – a family of XML-based reformulations of HTML 4.01.
XML
eXtensible Markup Language – a generic markup language for encoding documents in a machine- and human-readable format.
SGML
Standard Generalized Markup Language – an ISO standard for defining generalized markup languages, the ancestor of HTML and XML.
DOM
Document Object Model – a programming interface that represents HTML or XML documents as a tree of nodes.
W3C
World Wide Web Consortium – the standards organization founded by Tim Berners-Lee to develop open web specifications.
WHATWG
Web Hypertext Application Technology Working Group – a community of browser vendors and experts that maintains the HTML and DOM Living Standards.
IETF
Internet Engineering Task Force – an open standards organization that developed early Internet and web protocols, including HTML 2.0.
WAI
Web Accessibility Initiative – a W3C program that develops guidelines and resources to make the web accessible to people with disabilities.
I18n
Internationalization – shorthand for “Internationalization” (18 letters between the first “i” and last “n”), covering language and locale support.
CSS
Cascading Style Sheets – a language for describing the visual presentation of HTML and XML documents.
URI
Uniform Resource Identifier – a string that identifies a resource on the web, such as a URL.
URL
Uniform Resource Locator – a type of URI that specifies the location of a resource and the protocol to access it.

Key people

Tim Berners-Lee
Inventor of the World Wide Web and designer of the first HTML at CERN; founder and Director of the World Wide Web Consortium (W3C).
Robert Cailliau
Engineer at CERN who collaborated with Berners-Lee on early web proposals and helped promote the Web project.
Marc Andreessen
Lead developer of the Mosaic browser at the National Center for Supercomputing Applications (NCSA), co-founder of Netscape.
Dave Raggett
Computer scientist and W3C staff member who authored early HTML drafts, including HTML+ and HTML 3.0, and wrote about the history of HTML.
Ian Hickson
Principal editor of the HTML specification at the Web Hypertext Application Technology Working Group and an editor of W3C HTML5 drafts.
Lou Montulli
Developer associated with the Lynx text browser and early Netscape work, participant in early web standardization.
