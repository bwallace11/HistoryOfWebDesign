Glossary of HTML Terms

Hypertext

Text that links to other documents, enabling connected navigation across pages.

Markup

Structured annotations inside text that describe document meaning and organization.

Element

A document unit such as heading, paragraph, list, or link represented by tags.

Attribute

Configuration metadata on an element, such as href, src, or lang.

DTD

Historical document grammar used with earlier HTML generations before HTML5 parsing rules.

SGML

The standards family from which early HTML was originally derived.

W3C

The standards organization that published major HTML recommendations through multiple eras.

WHATWG

The group maintaining the HTML Living Standard and continuous specification updates.

Living Standard

A continuously updated standard that evolves with implementations and interoperability tests.

Semantic HTML

Meaningful element usage that improves structure, accessibility, and machine interpretation.
