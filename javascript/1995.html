1995 – The Birth of JavaScript

In May 1995, Brendan Eich created JavaScript at Netscape Communications in approximately ten days. The World Wide Web at that time consisted almost entirely of static HTML documents. Interaction required server requests, and user input triggered full page reloads. JavaScript introduced the radical concept of client-side execution — code running directly in the browser without contacting a server.

Originally named Mocha, then LiveScript, the language was renamed JavaScript shortly before release to align with the marketing momentum of Sun Microsystems’ Java. Despite the branding association, JavaScript was never a subset of Java. Its design combined influences from Scheme (functional patterns), Self (prototype inheritance), and C-style syntax.

Technical Characteristics

  • Prototype-based object model
  • Dynamic typing
  • First-class functions
  • Global scope by default
  • Direct DOM manipulation

There was no formal standard in 1995. Each browser implemented its own behavior. The language was designed quickly, without anticipating the scale of modern web applications.
