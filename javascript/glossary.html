Glossary of JavaScript Terms

ECMAScript

The formal specification name for the JavaScript language (ECMA-262).

TC39

The standards committee that evolves ECMAScript through staged proposals.

Prototype Chain

JavaScript’s inheritance mechanism where objects delegate behavior to parent prototypes.

Transpiler

A build tool that converts modern syntax into compatible output for older engines.

Strict Mode

An opt-in execution mode that tightens semantics and exposes silent errors.

DOM

The object representation of an HTML document used for runtime interaction.

Promise

An object representing eventual completion or failure of an asynchronous operation.

async/await

Syntax built on Promises for readable asynchronous flow.

JScript

Microsoft’s historical JavaScript implementation that contributed to compatibility fragmentation.

Polyfill

Runtime code that adds missing platform features in environments lacking native support.
